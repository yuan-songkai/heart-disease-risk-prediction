# Interpretable and uncertainty-aware heart disease risk prediction

## Overview
This project develops an interpretable and uncertainty-aware heart disease risk prediction model based on logistic regression.

The focus is not on maximizing predictive accuracy, but on:
- probabilistic risk estimation,
- interpretability,
- and uncertainty quantification for decision support.

## Dataset
UCI Heart Disease dataset(Cleveland subset).

### The dataset used in this project is the Heart Disease dataset from the UCI Machine Learning Repository (Cleveland subset), made available via Kaggle for convenience.

## methods
- Logistic Regression (baseline model)
- Feature standardization
- Odds ratios and coefficient interpretation
- Bootstrap-based uncertainty analysis

## Disclaimer
This project is for educational purposes only and should not be used as a diagnostic tool.
